Mi compromiso con este proyecto:

Quiero que cuando abramos el árbol de Pura 3, sientas que estás navegando la planta real.

No quiero que sea un software genérico.

Quiero que un técnico nuevo pueda aprender cómo está armada la línea simplemente recorriendo el árbol del sistema.

CMMS Pura no es un sistema para registrar órdenes de trabajo. Es una plataforma para administrar el ciclo de vida completo de los activos industriales y preservar el conocimiento del mantenimiento.
